# 📦 Supply Chain & Logistics Analytics Dashboard

## 🎯 Business Problem
Supply chain teams lack visibility into vendor 
reliability, delivery delays, and inventory health — 
leading to stockouts, SLA breaches, and revenue loss.

## 📦 Dataset
- 550 shipment records | 22 columns
- Vendors, Regions, Transport Modes
- Delivery Status, Defect Rates, Inventory Levels

## 🔧 Tools Used
| Tool | Purpose |
|------|---------|
| Microsoft Excel | Dashboard, Pivot Tables, Charts |
| SQL | 8 queries — vendor ranking, stockout alerts |
| Python | Dataset generation (pandas, openpyxl) |

## 📊 Key KPIs
- On-Time Delivery %
- Avg Delay Days  
- Vendor Score
- Fulfillment Rate %
- Defect Rate %
- Stockout Risk Count

## 💡 Key Findings
1. 3 vendors with OTD below 70% — flagged for review
2. Air transport most reliable, Road highest delays
3. 47 items below reorder point — stockout risk
4. Critical orders have disproportionate delay rate
